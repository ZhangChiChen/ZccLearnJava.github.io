# 什么是JDK、JRE

* JDK、JRE和JVM的包含关系

* 1. JDK = JRE + 开发工具集（例如javac，java编译工具等）

  2. JRE = JVM + Java SE标准类库（Java核心类库）

  3. JDK = JVM+ Java SE标准类库 + 开发工具集
  4. 代码开发人员需要安装JDK
  5. 如果只需要运行开发好的.class文件，只需要安装JRE
